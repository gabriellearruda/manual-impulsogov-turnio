# Manual ImpulsoGov · Turn.io

Manual interno para criação de chatbots no WhatsApp usando a plataforma [Turn.io](https://app.turn.io) e a API oficial do WhatsApp (Meta).

## Sobre

Este guia cobre todo o processo, do zero ao monitoramento em produção, e foi escrito para qualquer pessoa do time — independente de conhecimento técnico prévio.

**10 módulos:**

1. Conceitos Fundamentais
2. Pré-requisitos e Pontos de Contato
3. Configuração Inicial no Turn.io
4. Templates de Mensagem (HSM)
5. Construção do Fluxo do Chatbot
6. Testes
7. Publicação e Ativação
8. Monitoramento e Métricas
9. Manutenção e Iteração
10. Recursos e Links de Referência

## Como usar

Site estático de arquivo único (`index.html`). Abra diretamente no navegador ou faça deploy na Vercel.

## Deploy na Vercel

Importe o repositório em [vercel.com/new](https://vercel.com/new). Nenhuma configuração necessária.
